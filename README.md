# Rust Udemy

Curso de Rust Udemy. 
Codigo de ejemplo

## Installation

Instalar Rust desde la [página oficial](https://www.rust-lang.org/tools/install).

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Usage


En la carpeta raíz:
```bash
cargo build
cargo run
```
